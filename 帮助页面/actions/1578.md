# [](#header-2) 启动服务
**动作编号**: 1578  
**动作名称**: 启动服务  
**动作作用的对象**: 无条件  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 启动打印服务,系统服务管理(启动,停止,获取列表,获取服务信息)  
**动作内容**: run|"%B_Autohotkey%" "%A_ScriptDir%\外部脚本\系统设置\服务管理.ahk" "Spooler" "Start"  
以服务名称和控制服务的文本作为参数, 执行外部脚本文件 "服务管理.ahk", ATA 内置该动作  
**代码或详细解释**:  
参数1: 服务名称, 例如 Spooler 或 Print Spooler  
参数2: Start, 表示启动服务
