# [](#header-2) 窗口置顶
**动作编号**: 1049  
**动作名称**: 窗口置顶  
**动作作用的对象**: 任意窗口  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 将当前窗口置顶  
**动作内容**: canfunc|CF_AlwaysOnTop|%Windy_CurWin_id%  
以当前窗口的句柄为参数, 执行如意的内置的函数 "CF_AlwaysOnTop", 如意内置动作  
**代码及详细解释**:  
```Autohotkey
CF_AlwaysOnTop(WinId){
	WinSet, AlwaysOnTop,, ahk_id %WinId%
}
```
